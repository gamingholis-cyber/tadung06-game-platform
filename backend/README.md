# 🎮 TADUNG06 - Platform Game Online

Platform game online dengan sistem admin, integrasi E-wallet, dan manajemen user lengkap.

## Fitur
- ✅ Registrasi & Login
- ✅ Top-up Coin via E-wallet Dana
- ✅ 3 Jenis Game
- ✅ Admin Dashboard
- ✅ Live Chat 24/7

## Tech Stack
- Backend: Node.js + Express
- Frontend: React
- Database: MySQL
- Payment: Dana E-wallet API

## Setup
```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
